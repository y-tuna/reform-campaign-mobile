## Description

### Requirements
- Enable photo/receipt uploads with GPS and timestamp metadata capture
- Display proof status: pending, on hold, completed

### Implementation Details
- Use HTML5 Geolocation API to capture GPS and Date API for timestamp
- Integrate file input with preview and metadata tagging
- Render status badges using Ant Design Badge component

### Test Strategy
- Test metadata collection by mocking geolocation and checking metadata fields
- Upload sample files and verify status transitions
- Confirm UI updates on status change from mock API
